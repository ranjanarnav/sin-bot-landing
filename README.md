# 🤖 Sin Bot Landing Page

A modern and responsive landing page for **Sin Bot**, built to showcase commands, features, statistics, and bot information with a clean UI and smooth user experience.

---

## ✨ Features

- Modern Responsive Design  
- Commands Showcase  
- Features Page  
- Statistics Dashboard  
- About Section  
- Mobile-Friendly Layout  
- Interactive UI Components  
- Clean Folder Structure

## 🚀 Getting Started

### 1. Clone Repository

```bash

```

### 2. Open Project Folder

```bash
cd sin-bot-landing
```

### 3. Run Project

Open:

```txt
index.html
```

in your browser.
